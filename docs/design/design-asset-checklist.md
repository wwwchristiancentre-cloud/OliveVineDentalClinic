# Design Asset Checklist

## Required for Strong Design Output

### Doctor / Clinic Photography
- [ ] primary doctor portrait for hero
- [ ] secondary doctor/editorial portrait for about section
- [ ] optional clinic interior photo(s)
- [ ] optional treatment/environment imagery

### Brand Inputs
- [ ] confirmed logo files
- [ ] confirmed brand colors if different from current olive/gold direction
- [ ] typography confirmation if available

### Contact / Location
- [ ] confirmed address
- [ ] confirmed phone number
- [ ] confirmed WhatsApp number/link
- [ ] confirmed opening hours
- [ ] directions / map destination link

### Booking
- [ ] booking provider selected
- [ ] booking URL or integration method
- [ ] booking CTA label confirmation

### Trust / Proof
- [ ] approved testimonials
- [ ] review snippets or ratings if available
- [ ] trust badges, affiliations, or relevant certifications if available

### Content
- [ ] final homepage copy pass
- [ ] doctor bio / credentials summary
- [ ] short service descriptions
- [ ] privacy/legal page content source

## Nice to Have
- [ ] before/after imagery if permitted and appropriate
- [ ] emergency contact wording if applicable
- [ ] clinic philosophy quote from the doctor
- [ ] short FAQ answers

## Launch Minimum
If time is limited, the minimum viable design asset set is:
- one strong doctor portrait
- logo
- correct contact details
- correct booking link
- short service descriptions
- at least 2 approved testimonials or equivalent proof
- privacy page content source
