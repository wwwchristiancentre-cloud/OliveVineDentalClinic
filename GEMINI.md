# Project Context: Olive Vine Dental

## Overview
Premium 3D dental clinic website for Dr. Emmanuel Oke. "Pastor-Surgeon" vibe.

## Tech Stack
- Next.js 14, R3F, GSAP, CSS Modules, Sanity.io.

## Key Features (MUS)
- 3D IPC Shield Intro
- Morphing Dental Chair
- Crystal Jaw Experience
- Smart Scheduling (No Sun/Wed eve)
- Floating Tools
- Pastor-Surgeon Editorial
- Headless CMS (Sanity)
- Draco Compression

## Architecture
- `src/app`: Routes.
- `src/components`: React components.
- `src/features`: Complex features.
- `docs/mockups`: Source of truth for UI.
